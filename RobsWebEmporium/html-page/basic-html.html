Main Tags

<!DOCTYPE>

This is a document type declaration for the browser. To quickly input doctype with emmet abbreviation type "! + <tab>". DOCTYPE does not need a closing tag.

<html>

"html" is a root tag which holds all tag elements within it. Think of it as the global tag. You can specify the site language here with the attribute "lang = 'en'". Html needs closing tag </html>

<head>

"head" tags contain all meta data for the page. tags like <title>, <style>, <base>, <link>, <meta>, <script>, <noscript> can be placed here. Head tag needs closing tag </head>

<title>

The title tag defines/names the document. It must be text only, and is desplayed in the title bar, or tab name. It is a required tag in html


Page title tags names are crucial for search engines and the more descriptive the title the better off you are with you web-page. The title should be no longer then 50-60 characters because search only display about that many characters.


Do not use just a list of words as a title. Be descriptive. The Title tag support global attributes.

<body>

The "body" tag contain the pages content. Things that will generally be visible to the user. There can be only one body tag, even morticians know that. Body tags support both global and event attributes.
